第一阶段：预备实验

发表于 2022-05-09 更新于 2022-05-10

预备实验：关于次氯酸与酒精混用实际结果实验的报告

参与者：杨喆涵、施雨凡

一、材料：HClO 片（25mg/片）； 无水乙醇；84 消毒液

二、设计：

第一组 50mL 200mg/L HClO 加入 50 mL 无水乙醇

第二组 50mL 200mg/L HClO 加入 50 mL 75%乙醇

第三组 50mL 50mg/L HClO 加入 50 mL 无水乙醇

第四组 50mL 50mg/L HClO 加入 50 mL 75%乙醇

第五组 50mL 50mg/L 84 消毒液 加入 50 mL 无水乙醇

三、现象：

第一组迅速产生气泡，持续约 3-4s，液体保持无色，有明显的酒精的混合感（水和酒精折光

率不同的感觉），经施雨凡亲自闻有一股酒气味（与乙醇和 HClO 的味道不同），认为其可能

为乙醛

第二组实验现象与第一组无明显区别

第三组迅速产生气泡，持续不到 1s，液体保持无色。味道差别不大

第四组结果与第三组无明显区别

第五组实验迅速产生气泡，持续约 4s，液体颜色由绿变为浅绿。味道差别不大

四、结论：本实验两种消毒剂混合实验中，HClO 与乙醇分别采用最大浓度和平时使用浓度。

乙醇在整体实验中均处于过量情况，对实验结果无明显影响。在几次实验中速度都很快，冒气

泡时间长短与 HClO 的含量成正相关。反应后产生味道被我们认为是乙醛的气味（不排除我

们猜想有错误的可能，待查证）。第五次实验中，NaClO 的颜色变淡，由于 NaClO 本身无

色，应为加入乙醇稀释导致。无论在正常用量和最大用量上，将 HClO 与 NaClO 混合都会发

生反应。

资料查询：HClO 与酒精混用会发生反应，生成某些挥发性有机化合物，但只要按标准稀释

使用，对人体无较大影响

五、进一步猜想：针对这一次实验我发现 HClO 的浓度远小于乙醇，即使反应 并不会消耗大量乙醇，

混合液消毒效果有降低但仍可使用。后期将进行进一步实验。
